# A/B Test Calculator 📊

Statistical significance testing tool used
by product teams to make data-driven decisions.

## Live Demo
[Click here](YOUR_STREAMLIT_URL)

## Features
- Two-proportion Z-test with p-value
- 95% confidence interval visualization
- Normal distribution rejection region plot
- Sample size calculator with power curves
- Test duration estimator from daily traffic
- Multi-variant (A/B/C/D) analysis
- Pairwise significance comparison
- P-value simulator (interactive explainer)
- Test history with CSV export

## Tools Used
- Python, Streamlit, SciPy, Plotly, Pandas, NumPy

## How to Run Locally
pip install streamlit pandas numpy plotly scipy
streamlit run app.py
